# Adaptive Market Intelligence OS

A research-grade quantitative trading system for Indian equity markets,
combining test-time adaptive learning, causal inference, rough volatility
modelling, and multi-agent signal validation.

## Research Contributions
- First unified application of 2025-2026 ML techniques to NSE markets
- VPIN + Hawkes process microstructure layer for Indian equities
- AlphaAgent self-improving alpha loop on NSE mid/small cap universe
- GT-Score anti-overfit backtesting framework for emerging markets
